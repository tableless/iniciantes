Propriedade Font

Formatando fonts de texto

As propriedades font definem as características, das lestras em textos, titulos, etc..

Propriedades basicas para fontes

  • font-size: tamanho da fonte
  • font-style: estilo de fonte
  • font-weight: quanto mais escura a fonte é, tipo negrito
  • font-family: tipo de fonte
  • font: define todas as propriedades de fonte em uma declaração
  • color: define a cor da fonte
  • font-variant: fontes maiúsculas de menor altura

Exemplos

font-size

A propriedade font-size define o tamanho do texto, esse tamanho pode ser definido, com pixels.

p {
  font-size: 20px;
}

irá renderizar:

Sou um <p> com 20px de tamanho


Pode ser definido por em

p {
  font-size: 1.5em;
}

irá renderizar:

Sou um <p> com 1.5em de tamanho


Por porcentagem

p {
  font-size: 100%;
}

irá renderizar:

Sou um <p> com 100% de tamanho


Por letras

p {
  font-size: smaller;
}

irá renderizar:

Sou um <p> com smaller de tamanho

font-style

font-style define o tipo da fonte: italic, normal, oblique.

h1 {
  font-style: italic;
}

irá renderizar:

Sou um <h1> com italic de estilo 

h2 {
  font-style: normal;
}

irá renderizar:

Sou um <h2> com normal de estilo 

p {
  font-style: oblique;
}

irá renderizar:

Sou um <p> com oblique de estilo

Font-weight

A propriedade font-weight define a espessura da fonte

p {
  font-weight: bold; //deixará a fonte grossa
  font-weight: normal; //deixará a fonte como o própio nome já diz normal (padrão)
  font-weight: 900; //deixará a fonte de acordo com o numero 100 ... 900
}
font-family

A propriedade font-family é usada para definir uma lista de tipos de fontes.

O browser assume o primeiro nome que ele reconhece na lista.

h2 {
  font-family: arial, helvetica, serif;
}

irá renderizar:

Sou um <h2> com estilo de familia definido

font

Especifique todas as propriedades de fonte em uma declaração

p {
  font: italic bold 12px Georgia, serif;
}

irá renderizar:

Sou um <p> com apenas font declarado, e varias propriedades declaradas

font-variant

fontes maiúsculas "menores"

p.small {
    font-variant: small-caps;
}

irá renderizar:

Sou um <p> com small-caps
