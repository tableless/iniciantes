O que é HTML?

Linguagem base dos websites

HTML é a linguagem padrão de funcionamento de todas as páginas da web que são carregadas e interpretadas por aplicativos do tipo navegador de internet, como Firefox, Chrome, Safari e Internet Explorer. Ela se apresenta de forma simples, estruturada em arquivos do tipo texto (ASCII), sendo sua interpretação processada a fim de gerar conteúdo legível aos usuários, quer seja a simples apresentação de informações ou toda a elaboração visual de website ou sistema on-line.

O código-fonte, que estrutura o processamento a ser obtido em um dispositivo de leitura de conteúdo apropriado, sempre estará disponível ao usuário de alguma forma. Isso se fará através do uso de alguma funcionalidade do aplicativo. Como a web está disponível a uma série de dispositivos - de microcomputadores, passando por leitores de e-book, smatphones, etc - é preciso haver uma definição de como ela funciona, assim como o que e de que forma pode ser feito através dela.

O tratamento de todo o cliente web - entenda-se por cliente qualquer aplicativo ou dispositivo que seja capaz de ler e interpretar o código-fonte de um aplicação Internet apresentando uma saída legível ao usuário - segue o mesmo padrão. O código-fonte é interpretado, não havendo a necessidade de nenhuma compilação de HTML/XHTML, apenas a sua leitura e interpretação.

Resumindo: HTML é a linguagem de estruturação utilizada para criar aplicações web.
